# Briefly API

This project use AI for briefly text.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash

# development watch mode
$ npm run dev

# production mode
$ npm run prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
